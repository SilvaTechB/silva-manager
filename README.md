# Silva Manager

**Silva Manager** is an Android APK patcher and patch manager app — patch and manage your favourite Android apps with ease.

## About

Silva Manager is based on the work of [URV](https://github.com/Jman-Github/Universal-ReVanced-Manager) and [ReVanced](https://github.com/ReVanced/revanced-manager). It is built with modern Android tooling using Kotlin and Jetpack Compose.

- **Min Android version:** Android 8.0 (API 26)
- **Target SDK:** Android 15 (API 35)
- **Package name:** `app.silva.manager`

## Download

Get the latest release from the [Releases page](https://github.com/SilvaTechB/silva-manager/releases/latest).

## Building

### Requirements

- Android SDK 35
- JDK 17
- Gradle 8.14+

### Build

```bash
bash build-debug.sh
```

The debug APK will be output to `app/build/outputs/apk/debug/`.
